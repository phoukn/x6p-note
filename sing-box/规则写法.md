|---------------------------|----------------------------------------------------------------|--------------------------------------------------------------------------------|-------------------------|
| 完整域名精确匹配          | `"domain": ["www.baidu.com", "ipv6.baidu.com"]`                | 必须完全相等（区分 www. 前缀）                                                 | 常用                   |
| 域名后缀匹配（最推荐）    | `"domain_suffix": [".example.com", ".google.com", ".cn"]`      | **必须加点 `.` 开头**（否则不严格后缀匹配，官方要求）；直连/代理首选         | **最推荐域名分流**     |
| 域名关键词匹配            | `"domain_keyword": ["cdn", "api", "google"]`                   | 域名任意位置包含关键词（不区分大小写）                                         | 常用，但可能误杀       |
| 域名正则匹配              | `"domain_regex": ["^.+\\.google\\..+$", "^stun\\..+"]`         | 正则表达式匹配（性能稍差，慎用）                                               | 高级场景               |
| IP CIDR 网段匹配          | `"ip_cidr": ["192.168.0.0/16", "10.0.0.0/8", "fc00::/7"]`      | 支持 IPv4/IPv6 CIDR，局域网/私有常用                                           | 常用                   |
| 私有/局域网 IP 自动匹配   | `"ip_is_private": true`                                        | 自动匹配 RFC1918、RFC4193 等私有地址（推荐兜底）                               | 常用兜底               |
| GeoIP 区域匹配            | `"geoip": ["cn", "private"]`                                   | **已废弃（1.8.0 deprecated）**，1.12.0 将移除；需 geoip.srs 文件               | **过渡期用，尽快迁移** |
| Geosite 域名集匹配        | `"geosite": ["cn", "geolocation-cn"]`                          | **已废弃（1.8.0 deprecated）**，1.12.0 将移除；需 geosite.srs 文件             | **过渡期用，尽快迁移** |
| 规则集（现代主流方式）    | `"rule_set": ["geoip-cn", "geosite-google", "rule-adblock"]`   | **最推荐**！先在 route.rule_set 定义 tag，再这里引用；支持 remote/local .srs/.json | **当前 & 未来首选**    |
| 按进程/包名分流           | `"process_name": ["org.telegram.messenger", "com.twitter.android"]` | 仅 TUN 模式有效；Android 用包名，桌面用进程名                                  | Android/Windows 专用   |
